# Résolution de l'équation de la chaleur en 2D

## 1. Scolaire
Ce projet se fait dans le cadre de la matière CHPS0811 Programmation GPU avancée.

## 2. Sujet

## 3. Déroulement
* Version séquentielle
* Version OpenMP
